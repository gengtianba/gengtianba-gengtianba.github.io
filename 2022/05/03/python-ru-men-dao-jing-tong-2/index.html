一、for循环

（一）多次猜年龄

true_age = 48

for i in range(3):    
    guess_age = int(input('please guess a number: '))
    if guess_age > true_age:
        print('猜的太大了')

二、break和continue

用于控制循环流程，只能在循环中使用

（一）break

&emsp;&emsp;当程序遇到break，就会结束当前循环（如果是多层嵌套循环，只结束当前循环）

# 一共5层，每层9个房间，当遇到房间404时，不再打印4层房间号
for floor in range (1, 6):
    for room in range(1, 10):
        if floor==4 and room==4:
            break
        print(f'L{floor}-{floor}0{room}')

（二）continue

&emsp;&emsp;当程序遇到continue，就会结束本次循环，直接进入下一次循环

# 一共5层，每层9个房间，当遇到第三层时不打印房间号其它层都打印
for floor in range (1, 6):
    if floor == 3:
        continue
    for room in range(1, 10):
        print(f'L{floor}-{floor}0{room}')

三、while

&emsp;&emsp;与for循环不一样的是，while循环的次数可以是不定的，只要条件满足就可以一直循环下去。

# while版猜年龄，当猜到的年龄和真实年龄不符时，继续猜提示，猜到时退出。
true_age = 48
guess_age = 0
while guess_age != true_age:
    guess_age = int(input('please guess a number: '))
    if guess_age > true_age:
        print('猜的太大了')
    elif guess_age < true_age:
        print('猜的太小了')
    else:
        print('恭喜你，猜对了')
        break

五、京牌摇号小程序

（一）需求：

1、允许用户最多选三次
2、每次放出20个车牌供用户选择
3、京[A-Z]-[xxxxx]，可以是字母和数字的组合

（二）思路

1、车牌由两个部分组成，第一部分京+A-Z的随机数；第二部分五位字母和数字组合随机数
2、随机数的实现
（1）random模块

import random
random.choice('abcdefg')  # 参数也可以是一个列表

s = 'abcdefghijk'
random.sample(s, 3)  # 从数据源s中随机选三个值

random.randint(1, 100)  # 打印一个1到100的随机数

（2）string模块

import string

string.digits  #打印数字

string.ascii_letters  # 大写字母加小写字母
string.ascii_uppercase  # 大写字母
string.ascii_lowercase  # 小写字母

string.punctuation  # 打印特殊字符

（3）join方法

引号内为链接符，join接受列表，列表元素必须全部为字符串
'--'.join(['a', 'b', 'c','1','2'])

（三）代码

import random
import string

times = 0
while times < 3:
    times += 1
    license_plates = []
    for i in range(20):
        area = random.choice(string.ascii_uppercase)  # 选择A-Z其中一个作为地区
        part1 = '京' + area + '-'  # 拼接车牌第一部分

        num_str = string.digits + string.ascii_uppercase  # 车牌第二部分在A-Z和0-9数字中选择
        part2 = ''.join(random.sample(num_str, 5))  # 拼接车牌第二部分

        license_plates.append(part1 + part2)  # 拼接整个车牌
    print(license_plates)

    choice = input(f'please choice a license_plate  in the list \
    and you hanv 3 chance to choice a number: you have {3 - times} times left： ')
    print(str.center('', 80, '~'))

    if choice in license_plates:
        exit(f'your license_plates is {choice}')

六、综合联系–年会抽奖小程序

（一）需求

&emsp;&emsp;张三科技公司有300个员工，开年会抽奖，奖项如下:
一等奖 3名：泰国五日游
二等奖 6名：iPhone手机
三等奖 30名：羽毛球拍一副

（二）规则

1、共抽三次，第一次抽三等奖，第二次抽二等奖，第三次抽一等奖
2、每个员工限中奖一次

（三）代码

import random

employees = [employee for employee in range(300)]  # 一共300名员工
winners_each_round = [30, 6, 3]  # 抽3轮每轮中奖人数
times = 0


def choice_winner(participants, number_of_winners):  # 定义函数，实现抽奖并移除中奖人员
    winners = random.sample(participants, number_of_winners)
    print(f'中{times+1}等奖人员编号为：{winners}')
    for winner in winners:
        participants.remove(winner)


while times < len(winners_each_round):  # 进行抽奖,一共抽三轮
    choice_winner(employees, winners_each_round[times])
    times += 1
