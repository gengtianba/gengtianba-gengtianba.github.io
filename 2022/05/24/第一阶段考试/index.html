  4. 简述你理解的 ascii、unicode、utf-8、gbk 编码。

    编码就是一个密码本，建立了文字和数字之间的映射关系， ascii、unicode、utf-8、gbk是不同的编码系统
  - ascii编码：建立了英文和数字之间的映射关系，取值范围0~255，不支持中文
  - gbk 编码：建立了汉字（韩语、日语等亚洲语言）和数字之间的映射关系，一个中文字符由两个字节表示
  - unicode 编码：建立了世界上大多数语言与数字的映射关系，向下兼容ascii编码，由于其适宜性广，数据的存储一般使用该编码，但其数据量相对较大，不利于网络传输
  - utf-8 编码：本质上是对unicode编码的压缩，规定了对应的unicode码号使用多少字节进行表示，减少了数据量。

  26. 写代码实现，将IP转换为整数。

    如 10.3.9.12 转换规则为：
    10 00001010
    3 00000011
    9 00001001
    12 00001100
    再将以上二进制拼接起来，然后再进行一次翻转。

    最终将翻转之后的二进制转换为整型。

    '''
本文件实现功能为，将IP转换为整数。
如 10.3.9.12 转换规则为：
     10            00001010
      3            00000011
      9            00001001
     12            00001100
再将以上二进制拼接起来，然后再进行一次翻转。
最终将翻转之后的二进制转换为整型。
'''

ip_str = '10.3.9.12'
ip_list = ip_str.split('.')  # ['10', '3', '9', '12']

# 1、将ip_list按转换规则进行转换
bin_list = []
for item in ip_list:
    bin_ip = bin(int(item))  # '0b1010' ...
    data = bin_ip[2:].zfill(8)  # 00001010 ...
    bin_list.append(data)

# 2、再将以上二进制拼接起来，然后再进行一次翻转
result_reverse = ''.join(bin_list)  # 拼接
result_bin = result_reverse[::-1]  # 翻转

# 3、最终将翻转之后的二进制转换为整型
result = int(result_bin, base=2)
print(result)

  28. 写代码实现，数据格式化处理。

    text = """id,name,age,phone,job
1,alex,22,13651054608,IT 
2,wusir,23,13304320533,Tearcher
3,老男孩,18,1333235322,IT"""

# 将上述数据处理为如下格式的结果：
#    info = [{'id':'1','name':'alex','age':'22','phone':'13651054608','job':'IT'},.... ..]
# 提示：text的内容是根据 \n 分割（\n表示回车换行）。
    '''
本文件实现功能为，数据格式化处理。。
如   text = """id,name,age,phone,job
    1,alex,22,13651054608,IT
    2,wusir,23,13304320533,Tearcher
    3,老男孩,18,1333235322,IT"""

输出为：info = [{'id':'1','name':'alex','age':'22','phone':'13651054608','job':'IT'},
                .... ,
                ....
                ]
'''

text = """id,name,age,phone,job
          1,alex,22,13651054608,IT
          2,wusir,23,13304320533,Tearcher
          3,老男孩,18,1333235322,IT
        """

text_list = text.strip().split('\n')  # ['id,name,age,phone,job', '1,alex,22,13651054608,IT ', ...]
keys = text_list[0].split(',')  # ['id', 'name', 'age', 'phone', 'job']
values_list = text_list[1:]  # ['1,alex,22,13651054608,IT', ...]

info = []
for i in range(len(values_list)):
    info.append({})
    values = values_list[i].split(',')  # ['1', 'alex', '22', '13651054608', 'IT'] ...
    for j in range(len(keys)):
        key = keys[j].strip()
        value = values[j].strip()

        info[i][key] = value

print(info)

# 补充代码实现
    '''
本文件实现功能为，实现累乘计算器。
'''


while True:
    result = 1
    content = input("请输入内容(Q/q退出):")  # 用户可能输入 5*9*99.... 或 5* 9 * 10 * 99 或 5 * 9 * 99...
    if content.strip().upper() == 'Q':
        break

    num_list = content.split('*')  # ['5', ' 9 ', ' 10 ', ' 99']

    for num in num_list:
        num = int(num.strip())
        result *= num

    print(result)

    '''
本文件实现功能为，《棋牌游戏11点》。

- 生成一副扑克牌（自己设计扑克牌的结构，小王和大王可以分别用14、15表示 ）

- 3个玩家
  user_list = ["alex","武沛齐","李路飞"]

- 发牌规则
  - 默认先给用户发一张牌，其中 J、Q、K、小王、大王代表的值为0.5，其他就是则就是当前的牌面值。
  - 用户根据自己的情况判断是否继续要牌。
    - 要，则再给他发一张。
    - 不要，则开始给下个玩家发牌。
  - 如果用户手中的所有牌相加大于11，则表示爆了，此人的分数为0，并且自动开始给下个人发牌。

- 最终计算并获得每个玩家的分值，例如：
result = {"alex":8,"武沛齐":9,"李路飞":0}
'''
import random

# 1.生成一副扑克牌
cards_num = []
for i in range(1, 11):
    cards_num.append(str(i))

cards_num.extend(['J', 'Q', 'K'])  # 12345678910JQK
cards_col = ['♠', '♥', '♦', '♣']

cards_list = []
for color in cards_col:
    for index in range(len(cards_num)):
        # 将得分附到牌的后面 ['♠', '1', 1]
        if index > 9:
            cards = [color, cards_num[index], 0.5]
            cards_list.append(cards)
            continue
        cards = [color, cards_num[index], index + 1]
        cards_list.append(cards)

# 加入大小王
cards_list.extend([['大', '王', 0.5], ['小', '王', 0.5]])

user_list = ["alex", "武沛齐", "李路飞"]

result = {}

for player in user_list:
    # 每一个玩家，重置分数，打印提示语句
    score = 0
    print('hello,{}! welcome to our game'.center(50, '*').format(player))

    while True:
        # 给该玩家发牌，打印提示，加上分数，删除该牌
        draw_card = random.sample(cards_list, 1)[0]
        print('The cards you draw are:{}'.format(draw_card[:2]))
        score += draw_card[-1]
        cards_list.remove(draw_card)
        print(score)
        if score > 11:
            score = 0
            break

        # 询问是否继续
        user_input = input('do you want to continue: ')

        # 非法输入退出
        if user_input != '不要' and user_input != '要':
            exit('input wrong')

        # 输入不要结束本次循环
        if user_input == '不要':
            break

        # 输入要，抽排，计算得分，去掉抽到的牌
        else:
            continue

    result[player] = score

print(result)
