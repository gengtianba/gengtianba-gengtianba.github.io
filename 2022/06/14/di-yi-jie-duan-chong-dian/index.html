一、python介绍

（一）编程语言的分类

  • 翻译的维度

    • 解释型语言
    • 编译型语言

三、编码

编码，文字和二进制之间的一个对照表。

  • ascii编码：规定使用1个字节来表示字母与二进制的对应关系。

  • gbk编码：gb-2312编码，由国家信息标准委员会制作（1980年）。gbk编码，对gb2312进行扩展，包含了中日韩等文字（1995年）。在与二进制做对应关系时，有如下逻辑：

    • 单字节表示，用一个字节表示对应关系（ASCII码的内容）。2**8 = 256
    • 双字节表示，用两个字节表示对应关系（汉字）。2**16 = 65536种可能性。

  • unicode编码：unicode（对照表网址）也被称为万国码，为全球的每个文字都分配了一个码位（二进制表示）。

    • ucs2：用固定的2个字节去表示一个文字。
    • ucs4：用固定的4个字节去表示一个文字。
    • unicode的应用：在文件存储和网络传输时，不会直接使用unicode，而在内存中会unicode。

  • utf-8编码：包含所有文字和二进制的对应关系，全球应用最为广泛的一种编码（站在巨人的肩膀上功成名就）。本质上：utf-8是对unicode的压缩，用尽量少的二进制去与文字进行对应。

    具体压缩流程

    • 第一步：选择转换模板

        码位范围（十六进制）                转换模板
   0000 ~ 007F              0XXXXXXX
   0080 ~ 07FF              110XXXXX 10XXXXXX
   0800 ~ FFFF              1110XXXX 10XXXXXX 10XXXXXX
  10000 ~ 10FFFF            11110XXX 10XXXXXX 10XXXXXX 10XXXXXX
  
  例如：
      "B"  对应的unicode码位为 0042，那么他应该选择的一个模板。
      "ǣ"  对应的unicode码位为 01E3，则应该选择第二个模板。
      "武" 对应的unicode码位为 6B66，则应该选择第三个模板。
      "沛" 对应的unicode码位为 6C9B，则应该选择第三个模板。
      "齐" 对应的unicode码位为 9F50，则应该选择第三个模板。
       😆  对应的unicode码位为 1F606，则应该选择第四个模板。            

注意：一般中文都使用第三个模板（3个字节），这也就是平时大家说中文在utf-8中会占3个字节的原因了

    • 第二步：在模板中填入数据

      # 文字  ->  16进制编码号  ->  二进制
- "武"  ->  6B66  ->  110 101101 100110

四、数据类型

（一）整型（int）

整型其实就是十进制整数的统称，比如：1、68、999都属于整型。他一般用于表示 年龄、序号等。

1.1 定义：

number = 10
age = 99

1.2 独有功能：

int.bit_lenth()  #获取v1的二进制有多少个位组成。

1.3 公共功能：加减乘除

1.4 转换：

# int 转 其它
bool(0)  # 转布尔
str(1)  # 转字符串

# 字符串 转 int
int('12', base=10)  # 将字符串转为10进制整数，10为默认值可以不传
int('12', base=2)  # 将字符串转为2进制整数
int('12', base=8)  # 将字符串转为8进制整数
int('12', base=16)  # 将字符串转为16进制整数

（二）布尔（bool）

布尔值，其实就是 “真”、“假” 。

2.1 定义：

data = False
alex_is_sb = True

2.2 独有功能：

无

2.3 公共功能：加减乘除

print(True + True)  # 2
print(True - False)  # 1
print(True * True)  # 1
print(False / True)  # 0.0

2.4 转换：

整数0、空字符串、空列表、空元组、空集合、空字典、None转换为布尔值时均为False
其他均为True

2.5 其它：

如果在 if 、while 条件后面写一个值当做条件时，他会默认转换为布尔类型，然后再做条件判断。

（三）字符串（str）

字符串，我们平时会用他来表示文本信息。例如：姓名、地址、自我介绍等。

3.1 定义：

v1 = "包治百病"
v2 = '包"治百病'
v3 = """
吵架都是我的错，
因为大家打不过。
"""

3.2 独有功能（48）：

  1. str.startswith() 判断字符串是否以 XX 开头？得到一个布尔值

  2. str.endswith() 判断字符串是否以 XX 结尾？得到一个布尔值

  3. str.isdecimal() 判断字符串是否为十进制数？得到一个布尔值

    为什么不用isdigit()

    v1 = "123"
print(v1.isdecimal()) # True

v2 = "①"
print(v2.isdecimal()) # False

v3 = "123"
print(v3.isdigit()) # True

v4 = "①"
print(v4.isdigit()) # True

  4. str.strip()

    • 默认情况：去除字符串两边的 空格、换行符、制表符，得到一个新字符串
    • 传入字符串作为参数：去除字符串两边指定的内容
    • str.lstrip()：去除字符串左边指定的内容
    • str.rstrip()：去除字符串右边指定的内容

  5. str.upper() 字符串变大写，得到一个新字符串

  6. str.lower() 字符串变小写，得到一个新字符串

  7. str.replace() 字符串内容替换，得到一个新的字符串

  8. str.split() 字符串切割，得到一个列表

    • str.split('分隔符', n ) 只切遇到的前第n个

    • str.rsplit('分隔符', n ) 从右向左切

      file_path = "xxx/xxxx/xx.xx/xxx.mp4"

data_list = file_path.rsplit(".",1) # ["xxx/xxxx/xx.xx/xxx","mp4"]
data_list[1]  # 路径拓展名

  9. str.join() 字符串拼接，得到一个新的字符串

  10. str.format() 格式化字符串，得到新的字符串

  11. str.encode() 字符串转换为字节类型

    str.decode() 字节类型转换为字符串

  12. str.center() str.ljust() str.rjust()将字符串内容居中、居左、居右展示

  13. str.zfill() 帮助你填充0

3.3 公共功能：

  • 相加：字符串+字符串
  • 相乘：字符串*int
  • 包含运算：in
  • 求长度：len
  • 索引、切片、步长
  • 循环

3.4 转换：

一般情况下，只有整型转字符串才有意义。

3.5 其它：

字符串属于不可变类型

（四）列表（list）

列表（list），是一个有序且可变的容器，在里面可以存放多个不同类型的元素。

4.1 定义：

user_list = []
user_list.append("铁锤")  # 可变
user_list.append(True)  # 可变
print(user_list) # ["铁锤",123,True] 有序、可存放多种数据类型

4.2 独有功能

  • list.append() 追加，在原列表中尾部追加值。

  • list.extend() 批量追加，将一个列表中的元素逐一添加另外一个列表。

  • list.insert() 插入，在原列表的指定索引位置插入值

  • list.remove()在原列表中根据值删除（从左到右找到第一个删除）【慎用，里面没有会报错】

  • list.pop()在原列表中根据索引踢出某个元素（根据索引位置删除）

  • list.clear()清空原列表

  • list.reverse反转原列表

4.3 公共功能

  • 相加：两个列表相加获取生成一个新的列表。
  • 相乘：列表*整型 将列表中的元素再创建N份并生成一个新的列表。
  • 包含运算：in
  • 获取长度：len()
  • 索引、切片、步长
  • for循环

【面试题】 循环的过程中对数据进行删除

# 错误方式， 有坑，结果不是你想要的。

user_list = ["刘的话", "范德彪", "刘华强", '刘尼古拉斯赵四', "宋小宝", "刘能"]
for item in user_list:
    if item.startswith("刘"):
        user_list.remove(item)
        
print(user_list)

# 正确方式，倒着删除
user_list = ["刘的话", "范德彪", "刘华强", '刘尼古拉斯赵四', "宋小宝", "刘能"]
for index in range(len(user_list) - 1, -1, -1):
    item = user_list[index]
    if item.startswith("刘"):
        user_list.remove(item)
        
print(user_list)

# 实现原理：倒着取无论删不删后面的数据，都不会改变前面索引的顺序
# ["刘的话", "范德彪", "刘华强", '刘尼古拉斯赵四', "宋小宝", "刘能"]
# 5 "刘能" ["刘的话", "范德彪", "刘华强", '刘尼古拉斯赵四', "宋小宝"]
# 4 "宋小宝" ["刘的话", "范德彪", "刘华强", '刘尼古拉斯赵四', "宋小宝"]
# 3 '刘尼古拉斯赵四' ["刘的话", "范德彪", "刘华强", "宋小宝"]
# 2 "刘华强" ["刘的话", "范德彪","宋小宝"]
# 1 "范德彪" ["刘的话", "范德彪","宋小宝"]
# 0 "刘的话" ["范德彪","宋小宝"]

4.4 转换

  • int、bool无法转换成列表
  • str、tuple可以转换成列表

4.5 其它

  • 嵌套：列表属于容器，内部可以存放各种数据，所以他也支持列表的嵌套，对于嵌套的值，可以根据之前学习的索引知识点来进行学习

（五）元组（tuple）

元组（tuple），是一个有序且不可变的容器，在里面可以存放多个不同类型的元素。如何体现不可变呢？记住一句话：《”我儿子永远不能换成是别人，但我儿子可以长大”》

# 元组是不可变的，但元组的元素中嵌套有可变的数据类型时，元组中该元素内部可以改变

v1 = (True,123,"Alex",[11,22,33,44])
data = v1[-1]
data.append(3)

print(v1)  # (True, 123, 'Alex', [11, 22, 33, 44, 3])

5.1 定义：

议在元组的最后多加一个逗v1 = (“李杰”,”Alex”,)，用于标识他是一个元组。当元组只有一个元素时，元组的符号不会被当成运算符中的括号

d1 = (1)  # int 1
d2 = (1,) # tuple (1,)

              (1,2,3)

5.2 独有功能：

无

5.3 公共功能：

  • 相加：两个元组相加获取生成一个新的元组。
  • 相乘，元组*整型 将元组中的元素再创建N份并生成一个新的元组。
  • 成员运算：in
  • 获取长度：len()
  • 索引、切片、步长
  • for循环

5.4 转换：

其他类型转换为元组，使用tuple(其他类型)，目前只有字符串和列表可以转换为元组。

5.5 嵌套

由于元组和列表都可以充当容器，他们内部可以放很多元素，并且也支持元素内的各种嵌套。

（六）集合（set）

集合是一个 无序 、可变、不允许数据重复的容器。

一般什么时候用集合呢？ 就是想要维护一大堆不重复的数据时，就可以用它。比如：做爬虫去网上找图片的链接，为了避免链接重复，可以选择用集合去存储链接地址。

6.1 定义

v1 = { 11, 22, 33, "alex" }

注意：定义空集合时，只能使用v = set()，不能使用 v={}（这样是定义一个空字典）。

6.2 独有功能

  • set.add() 添加元素
  • set.discard() 删除元素
  • set.intersection() 交集
  • set.union ()并集
  • set.difference() 补集

6.3 公共功能

  • 计算补集：-
  • 计算交集：&
  • 计算并集：|
  • 计算长度：len()
  • for 循环

6.4 转化

int/list/tuple/dict都可以转换为集合其他类型如果想要转换为集合类型，可以通过set()进行转换，并且如果数据有重复自动剔除。

提示：这其实也是去重的一个手段。

6.5 其它

  • 集合的内部存储原理

  • 元素必须可哈希

    因存储原理，集合的元素必须是可哈希的值，即：内部通过哈希函数把值转换成一个数字。

    目前可哈希的数据类型：int、bool、str、tuple，而list、set是不可哈希的，因此集合的元素只能是 int、bool、str、tuple 。

  • 查找速度特别快

    因存储原理特殊，集合的查找效率非常高（数据量大了才明显）。

  • 容器比较

    是否可变 类型 是否有序 元素要求 是否可哈希 转换 定义空
    是 list 是 无 否 list(其他) v=[]或v=list()
    否 tuple 是 无 是 tuple(其他) v=()或v=tuple()
    是 set 否 可哈希 否 set(其他) v=set()

    注意：集合中的所有元素，包括元素里的子子孙孙都必须是可哈希的

（七）字典（dict）

字典是 无序、键不重复 且 元素只能是键值对的可变的 容器。

7.1 定义

v1 = {}
v2 = dict()
info = { 
    "age":12, 
    "status":True,  
    "name":"wupeiqi",   
    "hobby":['篮球','足球']  
}

字典中对键值得要求：

  • 键：必须可哈希。 目前为止学到的可哈希的类型：int/bool/str/tuple；不可哈希的类型：list/set/dict。（集合）

  • 值：任意类型。

7.2 独有功能

  • dict.get(, 返回值)获取值，键不存在，默认返回 None

  • dict.keys()所有的键

  • dict.values()所有的值

  • dict.items()所有的键值， 伪列表dict_items， 元素为元组(k1, v1) …

  • dict.setdefault(k, v)没有k则创建k，v 有k则保持原有k , v

  • dict.update({k: v})更新字典键值对, 没有k则创建k，v ,有k则更新k, v

  • dict.pop(k)移除指定键值对 返回值为v

  • dict.popitem()按照顺序移除（后进先出）

    • py3.6后，popitem移除最后的值。
    • py3.6之前，popitem随机删除。

7.3 公共功能

  • 求并集：|（Python3.9新加入）
  • 成员运算：in
  • 计算长度：len()
  • 索引：通过k
  • for 循环

7.4 转换

v = dict( [ ("k1", "v1"), ["k2", "v2"] ] )

print(v) # { "k1":"v1", "k2":"v2" }
info = { "age":12, "status":True, "name":"武沛齐" }

v1 = list(info)        # ["age","status","name"]

v2 = list(info.keys()) # ["age","status","name"]

v3 = list(info.values()) # [12,True,"武沛齐"]

v4 = list(info.items()) # [ ("age",12), ("status",True), ("name","武沛齐") ]

7.5 其它

  • 存储原理

  • 速度快

  • 嵌套

    • 字典的键必须可哈希，且它的子子孙孙都必须是可哈希的。（list/set/dict不可哈希）。
    • 字典的值可以是任意类型。
    • 字典的键和集合的元素在遇到 布尔值 和 1、0 时，需注意重复的情况。
    • 元组的元素不可以被替换

（八）None

Python的数据类型中有一个特殊的值None，意味着这个值啥都不是 或 表示空。 相当于其他语言中 null作用一样。

在一定程度上可以帮助我们去节省内存。例如：

# v1, v2指向同一块id

v1 = None  
v2 = None

# v3, v4指向不同的id

（九）浮点型（float）

浮点型，一般在开发中用于表示小数。

  • 在类型转换时需要，在浮点型转换为整型时，会将小数部分去掉。

  • 想要四舍五入保留小数点后N位：round()

  • 浮点型的坑（所有语言中）
