  1. 一个大小为100G的文件 etl_log.txt，要读取文件中的内容，写出具体过程代码。

    答：

    """一次读1/10"""
def read_file(file_path, read_times):
    with open(file_path, mode='rt', encoding='utf-8') as file_object:
        total_line = len(file_object.readlines())
        file_object.seek(0)
        lines_each_time, reminder = divmod(total_line, read_times)
        if reminder:
            read_times += 1

        i = 0
        while i < read_times:
            data_list = []
            for j in range(lines_each_time):
                data_list.append(file_object.readline())
            yield data_list
            i += 1


gen_obj = read_file('etl_log.txt', 10)

import os

file_size = os.path.getsize("etl_log.txt.txt")

    read_byte = 0
    while read_byte <= file_size:
        file_object.read(1)
        read_byte += 1

  5. 简述深浅拷贝？

    ​ 答：

    • 深浅拷贝只对可变类型有意义，不可变类型本质都不会对数据进行拷贝。
    • 不可变数据类型的浅拷贝：只对数据不可变类型的第一层进行拷贝，而不会对不可变数据类型和更深层级的可变类型进行拷贝。
    • 不可变数据类型的深拷贝：对所有层级的可变数据类型进行拷贝，不会对不可变数据类型进行拷贝。
    • 特例：对元组进行拷贝时，会对内部可变元素进行拷贝。

# 写timer装饰器实现：计算fun函数执行时间，并将结果给 result，最终打印（不必使用datetime,使用time.time即可）。

@timer
def func():
    pass

result = func()
print(result)

答：忘记@functools.wraps()内的参数

import time
import functools


def timer(origin):

    @functools.wraps(origin)
    def inner(*args, **kwargs):
        start_time = time.time()
        origin(*args, **kwargs)
        end_time = time.time()
        res = end_time - start_time
        return res
    return inner

@timer
def func():
    time.sleep(10)

存在问题:

  • @functools.wraps()内的参数，为原函数

  • 装饰器不要改变原函数的返回值

  2. 什么是正则的贪婪匹配？或 正则匹配中的贪婪模式与非贪婪模式的区别？

    答：

    • 贪婪匹配：匹配到正则表达式后，继续往后匹配，以求得满足结果的最长字符串
    • 非贪婪匹配：匹配到正则表达式后停止匹配返回结果

  1. 编写函数，实现base62encode加密（62进制），例如：
内部维护的数据有：0123456789AB..Zab..z(10个数字+26个大写字母+26个小写字母)。
当执行函数：
	base62encode(1)，获取的返回值为1
	base62encode(61)，获取的返回值为z
	base62encode(62)，获取的返回值为10

答：

import string


def base62encode(value):
    data = string.digits + string.ascii_letters
    res = ''
    if value < 62:
        res = data[value]

    count = 1
    while value > 61:
        div, reminder = divmod(value, 62)
        if div:
            res = str(div) + data[reminder]
        value -= div*pow(62, count)

    return res

存在问题：

  • 可适当简化
  • 不要把62写死，通过len获取进制
思路：一直让输入的数值除以62取余数，以此取到的余数分别是每一位的索引位置。
